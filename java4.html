JavaScript Events and Popup Boxes (Interactive Lesson)

1. What Are Events?

JavaScript events are actions that happen in the browser — like clicking a button, moving the mouse, typing a key, or submitting a form. We use events to trigger JavaScript functions. :contentReference[oaicite:1]{index=1}

3. Alert Popup Box

The alert box displays a message that the user must acknowledge (click OK). Useful for warnings or important information. :contentReference[oaicite:2]{index=2}

4. Prompt Popup Box

A prompt box asks the user for input. The user can enter text or click Cancel. :contentReference[oaicite:3]{index=3}

5. Confirm Popup Box

A confirm box asks the user to confirm or cancel something (OK / Cancel). It returns true if OK is clicked, false if Cancel is clicked. :contentReference[oaicite:4]{index=4}
