LESSON 4

 

 -A CSS RULE

has two main parts: a selector, and one or more list of style declarations.

Basic format of a CSS style rule is as follows:

For Inline style rule:
<tag style="style decoration">
<tag style="prop1: valuel; prop2: value; prop3: value;>

For document-level and external style sheets:
selector {style declarations}
selector {prop1:value; prop2:value; prop3:value;}

-STYLE PROPERTIES

Properties are collected into six groups:
1. Fonts
2. Colors and backgrounds
3. Text
4. Boxes and layout
5. Lists
6. Tag classifications

 

-CLASSES

What are Classes?
Class – is a user-defined selector.
- used to apply a style rule to specific HTML tags
- can control individual HTML element
formatting
- is a reusable style definition that allows you to
apply the same set of formatting rules to
multiple HTML elements
Example: you can create a class only for the
paragraphs to which you have assigned the class.

 

-CLASS SELECTOR

What are element+class selector (element.classname)?
• A style applied to a particular set of elements.
• More specific, it apply styles only if the element and
class match.

Example:
p.note {color:blue; font-size:18px;}

note=classname
p=element

 

Why use Element + Class selectors?
Class selector (.classname)
→ reusable across different elements.
Element + Class selector (p.classname)
→ precise control over styling only one type of element.

 

-GROUPING SELECTORS

You can group selectors and combine it in one CSS file. Each selector should be separated by a
comma.

Sample code:

From:

  • h1 {color: blue;}
    p {color: blue;}
    p{background-color: black;}

To:

  • h1,p {color: blue;}
    p {background-color: black;}
