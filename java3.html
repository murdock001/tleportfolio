JavaScript Statements, Expressions & Operators

1. Statements

A JavaScript statement performs an action. Example: assigning a value to a variable.

// let x = 10; // This is a statement

2. Expressions

An expression produces a value. Example: 5 + 3 results in 8.

Click the button to see expression results.

3. Arithmetic Operators

These perform math:

  • + addition
  • - subtraction
  • * multiplication
  • / division
  • % remainder

Click to run arithmetic example.

4. Comparison Operators

Compare values:

  • == equal (value only)
  • === strict equal (value and type)
  • != not equal
  • > greater than
  • < less than

Click to see comparison results.

5. Logical Operators

  • && – AND
  • || – OR
  • ! – NOT
