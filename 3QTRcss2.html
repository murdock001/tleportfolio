CSS Text Formatting Examples

1. Text Indentation

The text-indent property moves the first line of a paragraph inward from the left margin.

This paragraph shows text indentation. The first line is moved 40 pixels to the right.

2. Changing Text Case

The text-transform property changes how text appears. It can convert text to uppercase, lowercase, or capitalize each word.

This sentence is displayed in uppercase using the text-transform property.

3. Changing Line Spacing

The line-height property controls the vertical space between lines of text. Increasing it makes text easier to read.

This paragraph demonstrates increased line spacing. The space between lines is larger than normal, making the text clearer and more readable.

4. Adding Background Color to Text

The background-color property adds color behind the text element.

This text has a light blue background applied to it.

5. Adding a Border with Set Properties

The border property sets the border width, style, and color. In this example, the border is 3px thick, dashed, and green.

This box shows a border applied around the text.
