JavaScript Script Placement and Basics

1. Where can you place JavaScript?

In the <head> section: Scripts here run even before the page is visible.

In the <body> section: Scripts here run as the elements load and can affect the page content.

External scripts: These are stored in a separate .js file and linked using src. This makes code reusable.

5. JavaScript Guidelines

  • Case sensitive — myVar and myvar are different.
  • White space is ignored within statements but helps readability.
  • Use comments to explain code:
// This is a single line comment
/* This is a multi‑line comment */
