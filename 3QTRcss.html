CSS Typography Lesson

This page demonstrates how to use font-variant, text-shadow, and word-wrap in CSS.

1. Font Variant

font-variant changes how letters appear. One common value is small-caps, which turns lowercase letters into smaller capital letters.

Example:

The quick brown fox jumps over the lazy dog

2. Text Shadow

text-shadow adds a shadow effect behind text. It uses values like: horizontal offset, vertical offset, blur, and color.

Example:

This text has a shadow effect

3. Word Wrap

overflow-wrap: break-word forces very long words to wrap instead of overflowing outside the container.

Example:

ThisIsAnExtremelyLongWordWithoutSpacesThatWouldNormallyOverflowButNowItWrapsInsideTheBox.
